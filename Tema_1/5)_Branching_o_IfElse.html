5) Branching o If Else

Created sábado 29 diciembre 2018

En Ruby, Branching se le conoce al uso de if/else, veamos el siguiente pseudo-código.

Código

#Ejemplo de AND
condition = false
another_condition = true
#if/else
if condition && another_condition
puts "Hello"
#execute some code
else
puts "Bye"
#execute some other code
end
puts "LA LA LA"


Ejemplo de OR



#Ejemplo de OR
condition = false
another_condition = true
#if/else
if condition || another_condition
puts "Hello"
#execute some code
else
puts "Bye"
#execute some other code
end
puts "LA LA LA"




Ejemplo de NOT FALSE OR



#Ejemplo de OR
condition = false
another_condition = true
#if/else
if !condition || another_condition
puts "Hello"
#execute some code
else
puts "Bye"
#execute some other code
end
puts "LA LA LA"



Ejemplo de ==


Código

name = Carlos
if name == "Carlos"

puts "Welcome to program [ #{name} ]
elsif name == "Jack"
puts "Welcome to program, Jack"
elsif name == "Evgeny"
puts "Go back to helping students., Evgeny"
